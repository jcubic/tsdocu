Simple tsdoc Markdown documentation generator

## Usage

```bash
npx -y tsdocu
```

## Config

```json
{
   "include": ["src/**/**.tsx?"],
   "exclude": ["**/*.test.tsx?", "**/*.spec.tsx?"]
}
```
